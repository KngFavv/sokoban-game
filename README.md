# Sokoban Game
A Java implementation of the classic Sokoban puzzle game developed as part of my university
coursework which i took and made it more advanced.

## Features
- Grid-based puzzle gameplay
- Custom graphics and sprites
- Movement animations
- Multiple levels
- Collision detection
- Interactive tiles
- Object pushing mechanics

## Technologies

- Java
- Object-Oriented Programming
- Java Graphics API
